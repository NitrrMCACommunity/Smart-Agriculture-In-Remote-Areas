Our Concept

Due to lack of connectivity from virtual world remote areas farming could not get the scientific techniques of agriculture which can increase the production. To connect remote areas with virtual world we used many technologies & merge them to provide scientific ways of agriculture.

ZigBee Radio Network

This was impossible to collect the farms data like Soil Moisture , Humidity , Temperature wind speed etc without using any local network. ZigBee Network cheap and easy to installed with Internet of The Things so this is the best choice to collect the farms data to some repository.
